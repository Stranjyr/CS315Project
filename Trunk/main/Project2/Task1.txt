We will computerize the following: Entering card information, storage and editing of user and company profiles, the validation of entered information, and report generation of entered information. The data will be stored in a database at chocoholics anonymous headquarters. Hardware requirements are a server to host the database, an email server to send reports, provider terminals to enter payment information, and user terminals to run interactive mode. Our system will provide output  and receive information from terminals. It will serve reports to the financial system. Users will receive and request reports, and be able to submit information.
